This first release makes MCP tools feel native in pi by providing one stable mcporter interface for discovery, schema inspection, and execution. It focuses on practical day-one usability: clearer failure hints, safer output handling, and a modular architecture that keeps future improvements fast and reliable.

## 🚀 Features

### Release pi-mcporter extension

Introduces the pi-mcporter tool so users can discover and call MCP tools from pi through one stable interface. Users can search available tools, inspect schemas, and execute calls with clearer auth/offline error hints and safe output truncation.
